Introduction

如何组织博客内容，我想了很久．最终我打算把这里定位成一个记录型的博客．文章的形式包括但不仅限于

  1. 存在疑惑时或者碰到重要内容时，对细碎知识点的具体探究和整理．
  2. 完成某一阶段/某一类/某一章知识的学习时，进行相应的总结．（总结意味着只会组织总的框图，并不会对细节过分的记录，对于看博客的人来说可能很不友好．）
  3. 每周日会将UCE-WeekPlan上的内容整合成一篇带有总结的文章．（人会忘记，记录每天的行踪是我长久以来一直存念的想法，现在终于可以付诸实践了．）
  4. 随笔

目前只打算把博客作为面向自我的记录工具，写分享型的技术文章还要再成长很长一段时间．
