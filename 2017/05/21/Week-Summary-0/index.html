总结

2017/5/5 Friday

  • 生活方式
    这两个星期比之前自律了很多，作息固定为7:00-23:00，除去必须花费的时间（上课吃饭写作业等）以及浪费的一些时间（刷知乎，发呆），大概有5-8个小时的学习时间。以前不上课，可利用时间都是以天为单位，按照以前的心态，如果有一个小时的可利用时间和十个小时才能完成的任务，那这一个小时恐怕就要被浪费了。现在对时间碎片的利用提高了很多。类似的，如果一大片代码初看时毫无思路，比起毫无意义的发呆，应该老老实实地去看每个函数的作用。感谢早起。

  • 万事开头难
    相比之前花了两个多星期才能勉强开始lab1，这两周连续完成lab2和lab3进步了很多。但这也在情理之中，前几周拆炸弹的时候也是类似的情况，从准备到拆完第一个炸弹可能花了一周的时间，而后面的五个炸弹一共才花了一天的时间。
    解决问题的过程中，能力增长的速度是指数级的。

  • 一些忧虑
    首先又是课的问题，这学期虽然上课了，但是学习模式固定为上课前一天补知识点+清算作业，然后第二天上课的时候继续进度。然而作业并不是每周都有，这导致我知识点的刷新周期和作业的间隔周期一样，负面效果是：进度刷新之前可能听不懂老师在讲什么。本质上来说其实没什么，因为最后结果都一样。但是如果因为知识断层而导致去上课却没能听懂会给心理造成了一定的压力。并且，我很担忧付出的这点时间会回报怎样的绩点。。。
